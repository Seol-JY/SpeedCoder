## 업데이트

업데이트 내용은 발표자료와 명세서에 반영하지 않습니다.

- 자동완성 도입
- HTTPS 적용 (종단간 암호화는 추후 적용할 예정)
- 반응형 디자인 적용

## 브랜치 전략

- main 브랜치는 항상 배포 및 사용 가능한 상태 유지
- develop 브랜치에서 개발 및 테스트 완료 후 PR
